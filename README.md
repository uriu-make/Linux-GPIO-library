# Linux-SPI-library
GPIO library of C++ for linux. \
To install
```sh
make install
```
To build examples.
```sh
make examples
```
### Examples
- **blink.cpp** : Blinking LED
- **Input.cpp** : Reading the GPIO status
- **event.cpp** : Detects changes in GPIO status
- **paralleltest.cpp** : Multiple pin simultaneous input/output
